Trading Game simulation 
Create a marble betting game: 
draw a marble from a bag (assume it's random) 
a bag has 10 marbles: 6 green and 4 red 
if you draw a green marble you win same amount that you bet, 
you draw a red marble you Lose amount you bet. 
marbles are replaced into bag after each round 
before each draw decide how much you will bet and enter it 
you start the game with 1000 gold pieces 
the number of rounds/draws should be variable 
if you lose half of your money game is aver 
show out data as you go along.
Bonus : 
replace one of the green marbles with a black marble which you win 10X and one of the red marbles with a white marble which you lose 5X
